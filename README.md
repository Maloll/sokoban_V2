# SAE 101 : Jeu Vidéo "Sokoban"

Ce projet consistait à développer un jeu Sokoban entièrement en C, tout en apprenant les concepts fondamentaux de la programmation.

### Fonctionnalités
* **Plateau de jeu** : Affichage avec rendu textuel (via printf).
* **Sauvegarde** : Enregistrement de l'état de la partie et de l'historique des mouvements.
* **Système Undo** : Possibilité de revenir en arrière en cas d'erreur.
